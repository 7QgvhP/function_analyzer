# 変更履歴 (Change Log)

本プロジェクト（C Function Analyzer）のすべての変更は、このファイルに記録されます。

## [1.6.0] - 2026-06-15
### 追加
* Webview 上の変数名をクリックした際、コード上の配列（例: `hoge[i]`）や構造体（例: `hoge.A`, `ptr->member`）全体のアクセス表現もカバーして強調表示（ハイライト）される機能を追加。

## [1.5.2] - 2026-06-14
### 修正
* 関数ポインタ変数（例: `math_func`）の宣言を正しくパースして「内部変数（ローカル変数）」に分類されるように修正。
* 関数ポインタによる呼び出しが「入力変数」に誤検出されてしまう不具合を修正。
* 関数ポインタによる呼び出しを「呼び出し関数」リストから除外。

## [1.5.1] - 2026-06-14
### 修正
* Webview からのフォーカス移動時に、エディタ上の強調表示（ハイライト）が即座にクリアされてしまう不具合を修正。
* Webview 操作時にアクティブエディタが見失われ、強調表示（ハイライト）が適用されない問題を修正（複数エディタから解析対象ファイルを正しく探索するように改善）。

## [1.5.0] - 2026-06-14
### 追加
* 解析結果画面（Webview）上の変数名・関数名をクリックした際に、コード上の該当シンボルを一時的に強調表示（ハイライト）し、画面中央へ自動スクロールする機能を追加。
* エディタ上でカーソルを移動させたりテキストを編集したりした際に、強調表示（ハイライト）が自動で解除される機能を追加。

## [1.4.1] - 2026-06-14
### 変更
* 解析結果画面（Webview）における関数名・マクロ関数名の表示デザイン（レイアウト・下破線・ホバーエフェクト等）を変数名と統一。

## [1.4.0] - 2026-06-14
### 変更
* 解析結果画面（Webview）の各分類見出しから英語表記（`(Inputs)` など）を削除。
* 解析結果画面（Webview）の表示順序を、「呼び出し関数」の直後に「マクロ関数」が来るように変更。

## [1.3.0] - 2026-06-14
### 追加
* 変数名または関数名がすべて大文字のとき、通常の入出力や呼び出し関数とは別の分類として「マクロ変数 (Macro Variables)」および「マクロ関数 (Macro Functions)」のセクションに表示する機能を追加。
* 上記の大文字マクロ分類機能のオン・オフを切り替える設定項目（`c-function-analyzer.classifyAllUppercaseAsMacros`）を VS Code 設定に追加。

## [1.2.0] - 2026-06-09
### 変更
* プロジェクト内の識別子・コマンドID表記を `c-function-analyzer` に統一。
* 解析結果画面（Webview）のレイアウトを、ウィンドウ幅に関わらず常に縦1列で表示するように変更。
* 不要な開発ドキュメント（`requirements.md`、`implementation_plan.md`）を削除。

## [1.1.1] - 2026-06-06
### 修正
* 配列引数（例: `int a[]`）に対して配列書き込み（例: `a[5] = 10;`）を行った際、入力変数として誤認識されたり、グローバル変数（出力）として誤検出されたりするバグを修正。
* ポインタ引数の型表記において、型名に `*` が二重に付与されてしまう問題（例: `int **`）を修正。

## [1.1.0] - 2026-06-06
### 変更
* 解析結果画面（Webview）のUIを、VS Codeの標準テーマに適合するシンプルなフラットデザインに変更。

## [1.0.1] - 2026-06-06
### 修正
* `web-tree-sitter` のインポート処理を CommonJS（CJS）向けに名前付きインポートへ修正し、`init is not a function` のエラーを解決。
* `web-tree-sitter` を `0.20.8` にダウングレードし、WASM（C言語パーサー）との ABI バージョン不一致によるロードエラーを解決。
* パッケージ（VSIX）内の不要な旧 WASM ファイル（`web-tree-sitter.wasm`）および開発用デバッグログを削除（パッケージサイズを約 288 KB へ削減）。

## [1.0.0] - 2026-06-05
### 追加
* 初期リリース（C言語関数の静的解析拡張機能）。
* `web-tree-sitter` による AST 解析の実装（引数・ポインタ出力・グローバル変数・直接関数呼び出しの抽出）。
* VS Code テーマ適応型の Webview 解析結果画面の提供。
* ショートカットキー `Ctrl + Alt + A` での起動サポートを追加（C言語エディタ用）。
