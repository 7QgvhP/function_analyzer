# C Function Analyzer

C言語の関数定義部分で実行すると、その関数の入出力変数や内部変数、直接呼び出している関数の一覧を美しい Webview 画面で可視化する VS Code 拡張機能です。

## 主な機能
* **入力変数 (Inputs) の抽出**: 値渡し引数、値読み取りのみのポインタ引数、および参照されているグローバル変数を抽出します。
* **出力変数 (Outputs) の抽出**: 戻り値、関数内で書き込み（代入）が行われているポインタ引数、および書き込みが行われているグローバル変数を抽出します。
* **内部変数 (Internal Variables) の抽出**: 関数内で宣言されたローカル変数をリストアップします。
* **呼び出し関数 (Called Functions) の抽出**: 関数内で直接呼び出されている関数（およびマクロ）の一覧を重複なく抽出します（関数ポインタ経由の呼び出しは除外されます）。

## 使い方
1. C言語ソースコード（`.c` や `.h`）を開きます。
2. 解析したい関数の **関数名がある行（シグネチャ行）** にカーソルを置きます。
3. ショートカットキー **`Ctrl + Alt + A`** を押すか、右クリックして **「関数を解析 (Analyze C Function)」** を選択します。
4. エディタが分割され、右側に解析結果が表示されます。
